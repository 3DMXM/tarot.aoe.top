# 宝剑 六

![](/tarot/images/s6.webp)

## 介绍

插着六把宝剑的船与上面撑篙的船夫和一对母女，向着远方向着光明未来的平静水域驶去，而船下的水汹涌起伏。象征伤害的的六把宝剑不能拔起，否则船会沉没。他们只能带着危险前行，与伤害结为旅伴。事情不很糟，但也不会很好。

船夫手中的黑色竹篙，黑色象征潜力，以为将来会发生什么事情，不要因自己以为的事会发生将自己困死了，驶过汹涌就是无边的平静。

### 含义
代表混乱之后，逐渐回复平静。
