# 宝剑 八

![](/tarot/images/s8.webp)

## 介绍

图中的女人被蒙眼缚手，困在剑阵之中。他伸不出手，看不见东西，不能自由的移动。她的家，在背后遥远的山上。宝剑八的象征是一种孤独无援的处境。但希望还是有的 ，背后暖和的家在召唤鼓舞着女人冲破险阻，奔向自由。宝剑八表明的是人所处的状态。

### 含义
自身受限制丧失个人能力。
